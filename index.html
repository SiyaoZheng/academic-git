Architecture

1 hook + 4 skills. No intermediate layers. GitHub is the database.

Hook (Stop + SessionStart) auto-commit.sh → wip snapshots + crash recovery + fire-and-forget push
Skills (Claude invokes directly) /begin → /create-issue → /commit ← /refine-issue
GitHub (SSOT) Issue body (immutable plan) + Issue comments (append-only changelog) + Checklist [x] (progress)

Workflow

From session start to PR merge — zero manual git commands.

Formal Commit

          
/commit skill — 1 commit = 1 checklist item

        

      

      

        Read Issue (SSOT) → analyze changes → Codex reviews staged diff → commit as type(#N/X): desc → check off item [x] on GitHub → push → check DAG for next unblocked item.

        When all items are [x]: auto-create PR with Closes #N → Codex reviews full diff → squash merge → return to main.

Issue Refinement

          
/refine-issue skill — mid-stream scope changes

        

      

      

        Scope changed? Need new items? Item no longer needed?
        Issue body is never modified. All changes are logged as timestamped, append-only comments.
        Claude reconstructs current truth by reading body + all comments sequentially.

            Append-only — never edit, never delete

7

        

          
Completion

          
All checklist items [x] — fully automatic

        

      

      

        Claude detects all items checked off → Codex reviews full diff → gh pr create with Closes #N → gh pr merge --squash --delete-branch → git switch main && git pull.

        No "done" signal from Adrian needed. Milestone deliveries get tags: email-YYYY-MM-DD, meeting-YYYY-MM-DD, conference-YYYY-MM-DD.
      

    


  







  

    
Anti-ADHD Safeguards

    
Structural guardrails that don't rely on human discipline.


    

      

        
Session Start Drift

        
/begin forces Issue + item selection via AskUserQuestion before any work

      

      

        
Mid-Task Tangent

        
Mandatory Issue + item parameter on every /commit — can't commit untracked work

      

      

        
Scope Creep

        
DAG enforces item ordering — only unblocked items startable

      

      

        
Forgot to Save

        
auto-commit.sh silently snapshots after every Claude response

      

      

        
Crashed / Closed Laptop

        
auto-commit.sh commits leftovers on next SessionStart

      

      

        
Never Says "Done"

        
Claude auto-judges: all checklist [x] = auto PR + merge
