## 读者顺序

1. 广义社会科学同行
2. 潜在 endorsement 对象

4. workflow / harness / tooling 读者

## 工作流原则

ScholarOS 吸收 solo Git workflow 的思想，但不照搬命令。它把个人研究组织成四层：公开稳定层、日常工作层、局部研究单元、远端保存层。这样，一个人也可以让研究历史保持可解释、可恢复和可教学。
